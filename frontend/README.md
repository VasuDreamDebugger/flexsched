# React + Vite

This template provides a minimal setup to get React working in Vite with HMR and some ESLint rules.

Currently, two official plugins are available:

- [@vitejs/plugin-react](https://github.com/vitejs/vite-plugin-react/blob/main/packages/plugin-react) uses [Babel](https://babeljs.io/) for Fast Refresh
- [@vitejs/plugin-react-swc](https://github.com/vitejs/vite-plugin-react/blob/main/packages/plugin-react-swc) uses [SWC](https://swc.rs/) for Fast Refresh

## Expanding the ESLint configuration

If you are developing a production application, we recommend using TypeScript with type-aware lint rules enabled. Check out the [TS template](https://github.com/vitejs/vite/tree/main/packages/create-vite/template-react-ts) for information on how to integrate TypeScript and [`typescript-eslint`](https://typescript-eslint.io) in your project.


🧠 Quantitative Aptitude (Maths side)

This is about numbers + speed + pattern recognition under pressure.

1. Arithmetic (Most Important – ~50% weight)

Core idea: real-world math + calculations

Percentages
Profit & Loss
Simple & Compound Interest
Ratio & Proportion
Averages
Time, Speed & Distance
Time & Work
Mixtures & Alligations

👉 If you’re weak here, everything collapses. This is the base layer.

2. Algebra

Core idea: forming equations and solving

Linear equations
Quadratic equations
Inequalities
Surds & Indices

👉 Shows up in slightly tricky problem-solving, not direct formula dumping.

3. Number System

Core idea: how numbers behave

Divisibility rules
HCF & LCM
Remainders
Cyclicity
Factors & multiples

👉 This is where interviewers test your “thinking depth”.

4. Geometry & Mensuration

Core idea: shapes + formulas

Lines, angles, triangles
Circles
2D & 3D mensuration (area, volume)

👉 Usually formula-based but twisted in questions.

5. Modern Math

Core idea: logic + counting

Permutations & Combinations
Probability
Set Theory
Logarithms

👉 Feels small but questions are high-quality and tricky.

6. Data Interpretation (DI)

Core idea: speed + calculation

Tables
Bar graphs
Pie charts
Caselets

👉 Not hard conceptually — but time pressure kills people.

🧩 Logical Reasoning

This is about pattern + structure + deduction.

1. Verbal Reasoning
Coding-Decoding
Blood Relations
Direction Sense
Seating Arrangement
Syllogisms
Statements & Conclusions
Assumptions

👉 Core skill: eliminate wrong options fast.

2. Non-Verbal Reasoning
Series (number/image patterns)
Analogies
Classification
Mirror & Water images
Paper folding/cutting

👉 Pure pattern recognition. Practice heavy.

3. Analytical Reasoning (High-level logic)
Puzzles (linear, circular seating)
Input-output
Data sufficiency
Logical games

👉 This is where toppers separate themselves.
